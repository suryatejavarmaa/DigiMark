# How to Get Your Segmind API Key (100 Free Calls/Day)

## Step 1: Visit Segmind
Go to: https://www.segmind.com/

## Step 2: Sign Up
- Click "Sign Up" or "Get Started"
- Enter your email and password
- Verify your email

## Step 3: Get API Key
1. After logging in, go to your **Dashboard**
2. Look for **"API Keys"** section in the sidebar
3. Click to view or create your API key
4. **Copy** the API key (it looks like: `SG_xxxxxxxxxxxxx`)

## Step 4: Add to .env File
1. Open file: `c:\Users\Asus\Downloads\Demoo\Digi Demo\auto demo\.env`
2. Add this line:
   ```
   SEGMIND_API_KEY=SG_your_actual_key_here
   ```
3. Save the file

## Step 5: Restart Server
Stop and restart your Node.js server for changes to take effect.
